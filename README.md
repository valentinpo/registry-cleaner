# 🛠️ Registry Cleaner Windows

Безопасная утилита для поиска и удаления битых ссылок, отсутствующих исполняемых файлов и мёртвых расширений проводника в реестре Windows. Не затрагивает системные настройки и критические параметры.

## ✨ Возможности
- 🔍 **3 режима сканирования**: Быстрый (5 веток) → Глубокий (8) → Максимальный (18 безопасных веток)
- 🌙 **Тёмная и светлая темы** интерфейса с мгновенным переключением
- 🔄 **Автообновление** через GitHub Releases (скачивание, замена, перезапуск)
- 🛡️ **Автоматический запрос прав администратора** при запуске
- 💾 **Создание и восстановление `.reg` бэкапов** перед очисткой
- ⚡ **Один файл `.exe`** (Python + Tkinter + PyInstaller, без зависимостей)
- 📜 **Подробный лог** с таймштампами и статусами операций

## ⚠️ Безопасность
Программа сканирует только пользовательские и служебные ветки автозагрузки, деинсталляции и расширений. Не трогает `SYSTEM\CurrentControlSet`, `SAM`, `SECURITY` и другие критические разделы. Рекомендуется всегда создавать резервную копию перед очисткой.
